# Modelo de Análisis Macroeconómico del Mercado

Este repositorio contiene un **modelo de análisis macroeconómico del mercado** desarrollado en Python dentro de un notebook de Google Colab. El objetivo de este proyecto fue crear una herramienta capaz de procesar datos financieros, evaluar indicadores clave y generar comparativas del comportamiento del mercado, permitiéndome explorar patrones, tendencias y puntos de inflexión relevantes para la toma de decisiones.


---

## Descripción del proyecto:

Este proyecto fue creado para:

- Explorar patrones y tendencias en datos económicos y de mercado
- Comparativa de eficiencia entre los modelos imprelemtados (regresión, forest y boosting)
- Integrar múltiples librerías de Python para estructurar un flujo de trabajo completo de análisis  

Trabajado en Google Colab para facilitar la ejecución, visualización de gráficos y experimentación. 

---

## Herramientas y Librerías:

El análisis se realizó utilizando:

- **Python**: lenguaje de programación principal
- **Pandas & NumPy**: procesamiento y manejo de datos
- **Matplotlib y Seaborn**: visualizaciones y análisis exploratorio
- **Datasets de indicadores macroeconómicos**: usado para comparar tendencias y métricas

Estas herramientas permiten realizar análisis estadístico y visualizaciones aplicadas al contexto económico y financiero.
